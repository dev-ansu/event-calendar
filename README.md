# CALENDÁRIO DE EVENTOS

Este um calendário de eventos simples usando apenas:

- HTML;
- CSS;
- JavaScript;

Os eventos são armazenados no local storage.

# Funcionalidades

- Adicionar um evento;
- Apagar um evento;
- Pesquisar um evento;
- Filtrar um evento pela data.
